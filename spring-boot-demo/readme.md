# Spring Boot REST Controller Project

## Overview

This project demonstrates how to build a REST controller using Spring Boot. 

### Features

- **Spring Boot 3.0+** for creating RESTful APIs.


## Prerequisites

Ensure you have the following installed:

- **Java 17**
- **Maven 3.8+**
